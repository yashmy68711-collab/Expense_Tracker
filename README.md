# GUI Expense Tracker (Python)

## About

A simple GUI Expense Tracker built using Python and Tkinter.
This application helps users add and manage daily expenses while tracking the total amount spent.

---

## ✨ Features

✔️ Add expense name and amount
✔️ View expense list
✔️ Automatic total expense calculation
✔️ Simple and user-friendly GUI

---

## 🛠️ Tech Used

* Python
* Tkinter

---

## 🚀 How to Run

1. Download the project
2. Open the Python file
3. Run the program
4. Enter expense details and track spending
